# Movie Rating Prediction with Python

## Project Overview

This project predicts movie ratings using Machine Learning and Data Analysis techniques in Python. The dataset contains information about movies such as genre, release year, votes, and other attributes.

The objective is to build a predictive model that estimates the rating of a movie based on its features.

---

## Dataset Features

The dataset includes the following features:

- Movie Title
- Genre
- Release Year
- Director
- Votes
- Duration
- Rating (Target Variable)

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Rating Prediction

---

## Machine Learning Models Used

- Linear Regression
- Random Forest Regressor
- Decision Tree Regressor

---

## Model Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

---

## Data Analysis Insights

- Popular genres tend to receive higher ratings.
- Movies with higher vote counts generally have more reliable ratings.
- Certain years show trends in movie popularity.
